Una vez explorados y preparados las variables con todos los datos almacenados en el dataset, procedemos a implementar algoritmos que procesaran y analizaran toda la información que le brindemos a través de modelos o también redes neuronales.

Para crear un modelo, podemos buscar modelos ya pre-entrenados o también crear uno desde cero, este modelo tendría que ser entrenado con la cantidad de datos que le proporcionemos, con la finalidad de poner probar el modelo con un algoritmo de clasificación, en donde buscamos que su porcentaje sea lo más alto posible, esto solo se pueda dar si el modelo fue previamente entrenado correctamente.
